DEFINICION DE HTML

  1. Las siglas de HTML significan "Lenguaje de Marcado de Hipertexto" y se centra principalmente en la elaboracion de paginas web
  2. Este codigo aparecio por primera vez en 1991 y la creo Tim Ber-ners-Lee y contaba con 18 elementos (13 actualmente conservados)
  3. La base del lenguaje HTML son las etiquetas, son fragmentos de texto encerrados en parentesis singulares "<>", estas se usan para describir algo en su pagina web

DEFINICION DE CSS

  • Es un lenguaje de marcas que proviene de las siglas "Hojas de estilo en cascada" y fue creada por el consorcio internacional W3C (World Wide Web Consortium) en el a?o 1996
  • Fue creado debido a la dificultad de personalizar el estilo de una pagina web y su reiterada replicacion de elementos (casi de linea a linea por lo general), asi facilitando el proceso de la misma
  • La ventaja que brinda CSS es que el creador puede realizar cambios esteticos de la web sin tener que modificar el contenido de la misma, lo cual resulta muy util

DEFINICION DE JAVASCRIPT

JavaScript es un lenguaje de programacion de alto nivel, interpretado y orientado a objetos, utilizado principalmente en el desarrollo web. Se usa principalmente para aportar dinamismo a las paginas web de manera que se puede interactuar con los elementos de la pagina web, el manejo de eventos, etc. Esto debido a que es un lenguaje complejo pero bastante flexible.
